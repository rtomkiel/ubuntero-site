#

A soluç﻿ão encontrada para instalá-lo no Ubuntu é através do Wine. Com o Wine em “ponto de bala” no seu micro, baixe o Phonemic Chart clicando aqui. Devidamente baixado, clique com o botão direito do seu mouse no arquivo e dê “Propriedades” – Aba “Permissões” – clique em “Permitir execução do arquivo como um programa”.
