Loja de Software do Ubuntu – Ubuntu 9.10 Alpha 6

#
Ao que tudo indica, o Ubuntu terá uma nova ferramenta para instalar, atualizar e desinstalar programas. Além das tarefas tradicionas relativas a instalação de programas, esta ferramenta permitirá a compra de programas, por este motivo a ferramenta se chama “Loja de Software”. A idéia é que esta ferramenta substitua o “Acionar/Remover programas”, “Gerenciador de pacotes Synaptic”, “Gerenciamento de Atualizações” e talvez até o Gdebi. Estas ferramentas estarão disponíveis nos repositórios mas deixarão de estar presente na instalação padrão.

Pidgin 2.6.1 com voz e vídeo

#
Saiu a atualização do Pidgin para a versão 2.6.1 para o Ubuntu 9.04. A grande mudança desta versão é a possibilidade de usarmos voz e vídeo em conversas pelo protocolo xmpp, isto inclui o google talk. Para termos esta atualização, é necessário adicionar o seguinte repositório na sua lista de respositórios: – deb http://ppa.launchpad.net/pidgin-developers/ppa/ubuntu jaunty main Depois disso é só atualizar e aplicar as atualizações disponíveis.
