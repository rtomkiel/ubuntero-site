#

Acesso remoto com TeamViewer

#
O Ubuntu vem, por padrão, com uma ferramenta para acesso remoto chamada vino-server. Esta opção pode ser acessada pelo menu “Sistema/Preferências/Área de trabalho remota”. É uma ótima alternativa quando estamos trabalhando na mesma rede da máquina a ser acessada, mas quando o acesso é “mais remoto” como pela internet, o trabalho fica sofrível. Isto acontece porque o vino é nada mais que um servidor VNC. Usuários de outros sistemas tem alternativas muito melhores para acesso remoto, como por exemplo “área de trabalho remoto” da microsoft e o logmein.
