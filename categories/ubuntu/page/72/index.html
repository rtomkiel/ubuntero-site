Acesso remoto com o FreeNX

#
Algumas pessoas falam que o FreeNX é uma espécie de evolução do VNC, eu discordo. As tecnologias utilizadas nas duas aplicações são totalmente diferentes e seus objetivos também são totalmente diferentes. Enquanto o VNC é uma simples aplicação que permite controlar Desktops remotamente, o FreeNX fornece um Desktop independente e totalmente funcional sem bloquear, para uma única sessão, o computador acessado. Para termos uma comparação melhor e facilitar a definição do programa, podemos dizer que o FreeNX tem as mesmas funcionalidades do Microsoft Terminal Server.
