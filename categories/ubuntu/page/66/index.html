Ubuntu torna-se o sistema padrão dos telecentros

#
Serpro customiza uma versão mais leve do sistema operacional, capaz de funcionar em máquinas com configuração simples. Com o Ubuntu, os computadores dos telecentros passam a aceitar atualizações e oferecer opções de acessibilidade para deficientes visuais. A partir dessa definição do Programa Serpro de Inclusão Digital, todas as novas máquinas doadas pela empresa utilizarão o Ubuntu. Os mais de trezentos telecentros já existentes serão atualizados de forma gradual, durante visitas técnicas.
