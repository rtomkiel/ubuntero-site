Aumentar o número de áreas de trabalho no Unity

#
Muita gente reclama da falta de opções de personalização do Unity e uma das personalizações é o número de áreas de trabalho que no Unity temos habilitados apenas 4. Para resolver este problema é muito simples. Instale o aplicativo CCSM via Central de programas do Ubuntu. Abra o programa pressionando e soltando a tecla “Super”, também conhecida como tecla “Windows”, e digite ccsm. Apenas um programa aparecerá e deverá ser aberto.

Publicando código-fonte em um servidor pastebin com o WebBoard

#
WebBoard é um utilitário para a publicação rápida de notas de texto e código-fonte em um servidor pastebin para depuração colaborativa. Pode funcionar como um aplicativo independente ou como um applet para o painel do GNOME. O programa é muito fácil de usar. Digamos que queira compartilhar com um amigo, um código-fonte de um programa através de um mensageiro instantâneo ou em um local qualquer na internet. Bom, fica um tanto desconfortável ter que enviar centenas de linhas de código.

Falha no Startup Manager no Ubuntu – e a solução: GRUB Customizer

#
