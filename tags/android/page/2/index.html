Ubuntu One no Android

#
Uma ótima opção para manter seus arquivos sempre perto de você, sem precisar levar um pendrive ou HD portátil é utilizar o poder da nuvem. Para isso o Ubuntu já oferece há um bom tempo o Ubuntu One. O Ubuntu One é um serviço de armazenamento e sincronização de arquivos na nuvem totalmente gratuito. O espaço oferecido de forma gratuita é de 5 GB, podendo ser aumentado com contratação de espaço adicional ou indicando novos usuários.

Ubuntu começa bem o ano

#
Primeiro dia útil do ano e somos surpreendidos por um contador no site do Ubuntu. Vários blogs levantaram qual seria o grande lançamento que estava por vir. Assistir no Youtube Agora sabemos que o grande anúncio é o Ubuntu Phone OS. Assim como já havíamos mostrado no Ubuntu for Android, o Ubuntu Phone precisa de um hardware moderno para rodar totalmente. Um processador ARM A9 e 1 GB de memória são necessários para rodar o sistema de maneira mais básica, mas um processador Quad-Core e 2 GB de memória são necessários para rodar em sua plenitude.
