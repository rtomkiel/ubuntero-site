Melhorias no Rhythmbox via PPA

#
Uma coisa que sempre reclamei do rhythmbox, gerenciador de músicas, foi o gerenciamento de podcasts. Na nova versão, disponível via PPA, esta funcionalidade teve uma leve melhorada, mas ainda há muito o que se aprender com o antigo player padrão do Ubuntu, o banshee. Algumas mudanças visuais também foram feitas, mas a melhor mudança foi a possibilidade de pesquisar por podcasts diretamente na biblioteca do iTunes, e para você ter esta nova versão no Ubuntu 12.

Opencast, episódio zero

#
