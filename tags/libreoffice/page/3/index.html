Menu Global no LibreOffice

#
Na instalação padrão do Ubuntu, o LibreOffice não vem com o menu global funcionando. Para fazermos isto manualmente devemos executar um único comando. Para isso abra o terminal pressionando Ctrl+Alt+t e digite o seguinte comando: sudo apt-get install lo-menubar Para mim, aparentou não funcionar ao abrir o LibreOffice pela primeira vez após o comando. Fechei o programa e abri novamente e o LibreOffice então abriu com o menu global habilitado.
