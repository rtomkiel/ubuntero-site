AMD64?

#
Uma dúvida que muito vi pela lista de discussão do Ubuntu e hoje apareceu novamente no twitter é sobre a compatibilidade da imagem AMD64 com processadores Intel. A confusão é de se entender, já que as duas empresas são concorrentes e o que funciona para uma pode não funcionar para outra, ainda mais quando uma imagem vem com o nome de apenas uma das empresas. Este artigo serve para sanar esta dúvida e será bem curto.
