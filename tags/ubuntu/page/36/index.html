Cliente para Twitter Polly

#
Demorei um pouco mas aí está o artigo sobre o cliente para Twitter Polly. O projeto é desenvolvido por Marcelo Hashimoto, sim, mais um brasileiro desenvolvendo software de qualidade. Para instalar o programa abra o nosso amigo terminal pressionando as teclas Ctrl+Alt+t e digite os seguintes comandos: sudo apt-add-repository ppa:conscioususer/polly-daily sudo apt-get update sudo apt-get install polly A utilização do programa é simples. Ao abrir pela primeira vez o programa já pede para configurar suas contas através de um código de verificação fornecido pelo site do twitter, acho que isto pode e deve ser melhorado.

Central de programas do Ubuntu Online

#
Se algum dia você não tiver um computador com Ubuntu instalado e alguém lhe perguntar se existe algum programa específico o que você faz? Agora você pode acessar a Central de programas do Ubuntu Online. A proposta da Central de Programas Online é permitir que programas sejam pesquisados e que informações dos programas possam ser consultadas de qualquer lugar em qualquer computador independentemente do sistema operacional. Se você estiver usando Ubuntu ao acessar a página, você poderá instalar os programas pelo botão que aparecerá ao visualizar os detalhes da aplicação.
