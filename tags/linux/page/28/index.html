Remover PPA

#
É inevitável ficarmos sem adicionar algum PPA em nossos Ubuntu, sempre tem algum software que não está nos repositórios oficiais ou que a versão não é a mais atualizada. O problema é que alguns PPA’s são experimentais e nem sempre o resultado é agradável e para remover nem sempre lembramos quais foram os pacotes adicionados. Para resolver o problema e voltar os pacotes para as versões originais dos repositórios oficiais, podemos utilizar o programa ppa-purge que está disponível nos repositórios oficiais do Ubuntu.
