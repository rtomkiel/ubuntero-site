#

Assistir no Youtube Agora sabemos que o grande anúncio é o Ubuntu Phone OS. Assim como já havíamos mostrado no Ubuntu for Android, o Ubuntu Phone precisa de um hardware moderno para rodar totalmente. Um processador ARM A9 e 1 GB de memória são necessários para rodar o sistema de maneira mais básica, mas um processador Quad-Core e 2 GB de memória são necessários para rodar em sua plenitude.
