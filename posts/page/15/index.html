#

Esta conta tinha poderes para publicar comunicados e o sistema vBulletin, sistema do fórum, permite que comunicados sejam postados em HTML sem verificação por código malicioso.

Ubuntu Edge Indicator

#
Quer acompanhar a quantas andam as doações para o Ubuntu Edge sem ficar entrando no site de tempos em tempos? Instale o Ubuntu Edge Indicator e seja feliz. sudo add-apt-repository ppa:cwayne18/ubuntu-edge-indicator sudo apt-get update sudo apt-get install ubuntu-edge-indicator Execute pelo terminal o ubuntu-edge-indicator ou pressione Alt+F2 e digite ubuntu-edge-indicator. O indicador irá aparecer próximo ao relógio. Sei que a dica não é nada de mais, mas para quem doou e está torcendo para chegar ao objetivo, fica sempre aos olhos.
